## 📌 About Email Spam Detection

**Email Spam Detection System** is a machine learning-based project that classifies emails as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP).

It helps in automatically filtering unwanted emails by analyzing text patterns and learning from labeled datasets.

This project demonstrates practical implementation of **machine learning in real-world communication systems**.
